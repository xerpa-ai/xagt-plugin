## Description

X-Agent is a "Perplexity for crypto" research surface with a Bloomberg-terminal feel. It clusters live narratives from public RSS, pulls real prices from CoinGecko, on-chain TVL from DefiLlama, and routes every LLM call through OpenRouter so a single key unlocks the entire model catalog.
